Parking-slot detection based on Nvidia Jetson TX2 platform (Tutor: Xuan Shao, 19145648630)

Introduction

In the field of ADAS, how to detect parking-slots using vision-based technologies is a key problem. In our previous work, we have developed a vision-based park-slot detection system. For detecting marking-points, we used YoloV2. For local image pattern classification, we used a customized CNN. All the training and testing are performed on Nvidia Titan X on a workstation. Such a system can be deployed on Nvidia Jetson TX2, but it was not optimized for TX2.

In this project your task is to re-implement the system using Nvidia Deep Learning solutions. Particularly, at the inference stage, you need to use TensorRT.
