End-to-end Parking-slot detection with Caffe and C++ (Tutor: Junhao Huang, 15221353227)

Introduction

In the field of ADAS, how to detect parking-slots using vision-based technologies is a key problem. In our previous work, we have developed a vision-based park-slot detection system. It is an end-to-end framework, implemented with Python and Pytorch. However, it is not optimized for on-vehicle computational platforms. In most cases, a software needs to be implemented with C/C++ to make it compatible with hardwares. In this project, your task is to port the python+pytorch based parking-slot detection system to a Caffe+C/C++ based one. 

Requirements

1> I will provide you a parking-slot detection system implemented with python+pytorch.

2> You task is to port it to a Caffe+C/C++ based one.
